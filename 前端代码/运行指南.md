# APICloud 项目运行指南

## 一、安装 apicloud-cli

```bash
npm install apicloud-cli -g
```

## 二、安装完成后的使用步骤

### 1. 进入项目目录

```bash
cd 前端代码
```

### 2. 登录 APICloud 账号

```bash
apicloud login
```

按提示输入：
- **邮箱**：您的 APICloud 账号邮箱
- **密码**：您的 APICloud 账号密码

### 3. 初始化项目（首次运行需要）

```bash
apicloud init
```

### 4. WiFi 真机同步（推荐用于开发调试）

#### 方式一：使用命令行启动 WiFi 同步服务

```bash
# 启动 WiFi 同步服务
apicloud wifiStart --port 8686
```

然后在 APICloud Studio 3 中或使用手机 AppLoader/自定义 Loader 连接到该服务进行同步。

#### 方式二：直接在项目目录启动 WiFi 同步

```bash
# 在项目根目录执行
apicloud wifiSync --project ./ --updateAll false --port 8686
```

### 5. 编译和打包

#### 编译自定义 Loader（用于真机调试）

```bash
# 编译 Android 自定义 Loader
apicloud build --type loader --platform android

# 编译 iOS 自定义 Loader
apicloud build --type loader --platform ios
```

#### 编译正式包

```bash
# 编译 Android 正式包
apicloud build --type package --platform android

# 编译 iOS 正式包
apicloud build --type package --platform ios
```

### 6. 其他常用命令

```bash
# 查看帮助
apicloud help

# 查看当前登录状态
apicloud info

# 退出登录
apicloud logout

# 查看项目信息
apicloud project info
```

## 三、开发流程

1. **开发调试**：
   - 在电脑上修改代码
   - 使用 WiFi 同步功能将代码同步到手机
   - 在手机上实时查看效果

2. **真机调试**：
   - 使用自定义 Loader 进行真机调试
   - 可以调用所有 APICloud 原生功能

3. **打包发布**：
   - 开发完成后，使用编译命令生成安装包
   - 上传到应用商店或直接分发

## 四、注意事项

1. **首次使用**需要登录 APICloud 账号
2. **WiFi 同步**要求手机和电脑在同一个 WiFi 网络下
3. **自定义 Loader**需要先在 APICloud 控制台创建应用，并获取 App ID（config.xml 中的 widget id）
4. **项目配置**请确保 `config.xml` 文件配置正确

## 五、快速开始示例

```bash
# 1. 进入项目目录
cd 前端代码

# 2. 登录（首次需要）
apicloud login

# 3. 启动 WiFi 同步服务
apicloud wifiStart --port 8686

# 4. 在手机上使用 AppLoader 或自定义 Loader 连接 WiFi 同步服务
#    输入电脑的 IP 地址和端口号（如：192.168.1.100:8686）
```

## 六、使用 APICloud Studio 3 IDE

也可以直接使用 APICloud Studio 3 可视化 IDE 进行开发：
- 下载地址：https://www.apicloud.com/devtools
- 支持可视化界面、一键同步、真机调试等功能
